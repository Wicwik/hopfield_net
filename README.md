# Hopfield neural network

## Requirements
- python3.x (where x > 6)
- \*nix OS

## How to run
- `git clone https://github.com/Wicwik/hopfield_net.git`
- `cd hopfield_net`
- `python -m venv venv_hopfield_net`
- `source venv_hopfield_net/bin/activate`
- `pip install -r requirements.txt`
- `python main.py`
